# 困難拉圖斯控時練習

非官方的《楓之谷》困難拉圖斯機制練習工具。  
重點在控時、走位、技能判斷、隊友協作與失誤演練。

## 專案連結

- 遊玩網站: https://hardpapulatus.2yangfam.com/困難拉圖斯控時練習
- GitHub: https://github.com/TzuMinYang/artale_hard_papulatus_simulator
- 贊助: https://portaly.cc/tmyang/support

## 主要功能

- 王血量起始模式（完整/控時/黑球/分身/黑水/紅鬧鐘/狂暴紅鬧鐘）
- 背後時鐘、黑球、反盾、紅藍鬧鐘、黑水、分身、玩家炸彈
- 法師指揮與物理控時流程（含延遲、沒聽 call、雜訊提示）
- 隊友移動與死亡邏輯（含回放與時間軸資訊）
- 設定頁可匯入/匯出 JSON

## 使用方式

1. 下載或複製本專案檔案。
2. 直接用瀏覽器開啟 `困難拉圖斯控時練習.html`。
3. 進入 `設定` 調整參數後開始遊玩。

## 檔案需求

請確認以下素材檔案與 HTML 在同一層目錄：

- `拉圖斯.png`
- `背景.png`
- `地板.png`
- `時鐘.png`
- `背景音樂.mp3`

## 設定匯入/匯出

- 在設定頁底部可使用 `匯出 JSON`、`匯入 JSON`
- 可保存目前模式、參數、鍵位與安全點設定，方便下次直接套用
